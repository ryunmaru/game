# Mini FPS (ブラウザゲーム)

シンプルなレイキャスティング方式で作った、1ファイル構成のFPS風ゲームです。

## 遊び方

1. サーバーを起動する
2. ブラウザで `http://localhost:8000` を開く
3. 画面をクリックして開始
4. 敵を倒してスコアを伸ばす

### 操作

- `WASD`: 移動
- `マウス`: 視点移動
- `クリック`: 射撃
- `R`: ゲームオーバー後にリスタート

## ローカル起動

### Node.js を使う（推奨）

```bash
npm run dev
```

`PORT` が設定されていればその値を利用し、未設定時は `8000` で起動します。

### Python を使う（代替）

```bash
python3 -m http.server 8000
```

## Not found になる場合

- 必ずサーバーを起動してからアクセスしてください。
- ルート以外のパスにアクセスしても `index.html` を返すようにしてあるため、プレビュー環境でも表示しやすくなっています。
