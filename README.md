# Mini FPS (ブラウザゲーム)

シンプルなレイキャスティング方式のFPS風ゲームです。PC/スマホ両対応です。

## 遊び方

1. サーバーを起動する
2. ブラウザで `http://localhost:8000` を開く
3. クリック/タップして開始
4. 敵を倒してスコアを伸ばす

## 操作

### PC
- `WASD`: 移動
- `マウス`: 視点移動
- `クリック`: 射撃
- `R`: ゲームオーバー後にリスタート

### スマホ / タブレット
- 左スティック: 移動
- 右スティック: 視点移動
- `FIRE` ボタン: 射撃
- ゲームオーバー時に画面タップで再開

## ローカル起動

### Node.js を使う（推奨）

```bash
npm run dev
```

### Python を使う（代替）

```bash
python3 -m http.server 8000
```
